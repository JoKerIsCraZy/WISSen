# data/

Runtime volume — not checked into git. Files generated by the app:

- `wissen.db` — SQLite database (grades, schedule, history, push subscriptions). **Backup this.**
- `settings.json` — user-editable runtime config (mode 0600). **Backup this.**
- `.api-token` — auto-generated API token if `API_TOKEN` env not set (mode 0600).
- `vapid.json` — auto-generated VAPID keys for Web-Push (mode 0600). Persist this; regenerating it invalidates all existing push subscriptions.
- `storage.json` — Playwright browser session cache (mode 0600). Speeds up scrapes by reusing the MS-SSO session. Safe to delete; will be recreated.
- `.weekly-detail-at` — timestamp of last weekly detail-refresh.

When running in Docker, mount this directory as a volume (e.g. `./data:/app/data`).
